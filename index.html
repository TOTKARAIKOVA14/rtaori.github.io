Projects

Here are some of my favorite projects! Check out my Devpost for more on what I make.

Rekt: Your Collision Companion

Rekt is an iOS app that streamlines the process for flustered drivers in an accident. It provides a convenient way to contact emergency, store crash information and pictures, exchange contact information with others, and more. 1st Place Winner at OHacks.

  • See the Post

Messenger Chat

Messenger Chat is a full-fledged Java desktop messenger: one can send messages, pictures, and audio to everyone connected to the current server. It even provides the ability to download a compressed conversation history (it uses the LZW algorithm!).

  • Watch a Demo
  • See the Code

Drone FPS

DroneFPS is a virtual reality game that uses a drone to provide an immersive experience. My team hacked Parrot AR.Drone 2.0, controlling it with a PS3 controller and streaming its video feed so that we could overlay it with a zombies to create a VR first-person shooter.

  • Watch a Demo
  • See the Code

Candidate Catch

This bucket-ball game of political trivia increases political awareness among younger generations by "gamifying" politics just in time for the 2016 elections. Users win by catching the candidate who meets the qualifying statement.
